# Kajabi Design Team

AI-powered tools for the Kajabi design team.

## Quick Start

### One-time setup (5 minutes)

1. **Download Claude Code Desktop** (if you don't have it)
   - Go to [claude.ai/download](https://claude.ai/download)
   - Download and install Claude for Desktop
   - Sign in with your Kajabi enterprise account

2. **Clone this repo**
   - Open GitHub Desktop (download from [desktop.github.com](https://desktop.github.com) if needed)
   - File → Clone Repository
   - Paste the repo URL and clone to your computer
   - Remember where you saved it (e.g., `Documents/kajabi-design-team`)

3. **You're done!**

### Running a design review

1. Open **Claude Code Desktop**
2. Click the **Code** tab (not Chat)
3. Click **Open folder** and navigate to your `kajabi-design-team` folder
4. Type `/design-review` and press Enter
5. Follow the prompts - Claude will ask you questions about your work
6. When done, you'll have a markdown document in the `design-reviews/` folder

### Sharing screenshots

When Claude asks for screenshots, you can:
- **Drag and drop** images directly into the chat
- **Paste** from clipboard (Cmd+V after taking a screenshot)
- **Reference Figma** by sharing a link or taking screenshots of your frames

### Getting updates

When the team updates the design review skill:
1. Open GitHub Desktop
2. Click "Fetch origin" then "Pull"
3. That's it - next time you run `/design-review`, you'll have the latest version

## Tips

- **Week 2 reviews** are forward-looking. Claude will help you think through what to watch for.
- **Week 4 reviews** are full audits. Bring screenshots of the actual build if possible.
- **Exploratory reviews** work too - for auditing existing flows or early ideas outside a cycle.
- The output document is meant to help frame discussion in design review meetings - bring it with you!
